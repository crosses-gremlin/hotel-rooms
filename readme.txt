== Description ==

1. Upload the entire `hotel-rooms` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **Hotel Rooms** menu in your WordPress admin screen.
